# 📝 NotApp

Modern ve kullanıcı dostu bir not alma uygulaması. Flutter ile geliştirilmiş, hızlı ve güvenilir bir deneyim sunar.

## ✨ Özellikler

- **🌟 Şık Tasarım**: Derin mor renk paleti ile estetik bir arayüz.
- **📝 Not Ekleme**: Başlık, içerik ve tarih ile not oluşturun.
- **🔍 Akıllı Arama**: Başlıklara göre anlık arama yapın.
- **📅 Tarih Filtreleme**: Notları yeni veya eski tarihe göre sıralayın.
- **🗑️ Kolay Silme**: Uzun basarak notları silin.
- **⚡ Hızlı Performans**: Isar veritabanı ile hızlı veri işlemleri.
- **📱 Responsive**: Mobil cihazlarda mükemmel görünüm.

## 🚀 Kurulum

### Gereksinimler
- Flutter SDK (3.0+)
- Dart SDK
- Android Studio veya VS Code

### Adımlar
1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/yourusername/notapp.git
   cd notapp
   ```

2. Bağımlılıkları yükleyin:
   ```bash
   flutter pub get
   ```

3. Uygulamayı çalıştırın:
   ```bash
   flutter run
   ```

## 📖 Kullanım

1. **Başlangıç**: Uygulama açıldığında hoşgeldiniz ekranı görünür.
2. **Not Ekleme**: Sağ alt köşedeki + butonuna tıklayın, başlık, içerik ve tarih girin.
3. **Arama**: Üst çubuktaki 🔍 ikonuna tıklayın, başlık ara.
4. **Filtreleme**: Üst çubuktaki 📋 ikonuna tıklayın, tarih sıralamasını seçin.
5. **Silme**: Notlara uzun basın ve silmeyi onaylayın.

## 🛠️ Teknolojiler

- **Flutter**: UI framework
- **Dart**: Programlama dili
- **Isar**: NoSQL veritabanı
- **Path Provider**: Dosya yolu yönetimi
